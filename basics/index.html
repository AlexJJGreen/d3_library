Scales and Axes

As the core concepts and components of D3, Scales and Axes require a fundamental understanding.

Scales

Scales are functions that take an input (from the domain) and produce an output (in the range). The common misconception is that the domain and range represent the X and Y axis, this is NOT correct! The domain represents the potential values of your data, the range represents how those will be displayed as pixels on the screen.

Domain

The domain is the set of possible values for your scale. For example, the domain of a linear scale in a test graded from 100 would be [0,100].

In an ordinal scale of fruits, the domain may be [apple, banana, melon].


Range

The range then is the possible set of outputs produced by a scale.

We can think of this like any mathematical forumala and graph. Essentially the domain is the potential inputs (x), the scale is the formula or function for that input value (f(x)) and the range is the potential output values of that (y).

What this means is simply, when given a value from the defined domain, D3 converts it to the given range.
